# net-test

> Test the quality of your wifi (using [speedtest.net](http://www.speedtest.net)) and get the results saved for future reference.

![](screenshot.gif)


## Install

```
$ npm install --global net-test
```


## Usage

```
$ net-test --help

  Usage
    $ net-test

  Options
    --no-parse  Skip uploading results to parse
```
